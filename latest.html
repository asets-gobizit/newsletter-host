Zoho CRM

Advanced Cross-Module Filtering Now Available

Zoho CRM has introduced cross-module filtering capabilities that allow users to create more sophisticated data views. This enhancement enables filtering of custom views and advanced filters using fields from related lookup modules, streamlining data analysis across connected records.

 
Zoho Billing

AI-Powered MCP Server Integration

Zoho Billing now features an AI-powered MCP (Model Context Protocol) server that automates billing processes and enhances reporting capabilities. This intelligent system can handle routine billing actions automatically while providing advanced analytics for better financial insights.

 
Zoho Deluge

Native PDF Conversion Support Added

Zoho's Deluge scripting platform now includes built-in PDF conversion functionality, eliminating the need for external services. This native support simplifies document generation workflows and enables developers to create PDFs directly within their automation scripts.

 
Zoho CRM

Enhanced Zia Forecasting with Performance Insights

Zoho CRM's forecasting module has been upgraded with advanced Zia intelligence that provides detailed explanations for missed targets. The AI now offers actionable suggestions for performance improvements, helping sales teams understand and address forecasting gaps more effectively.

 
Zoho CRM

Zia Dashboard Analytics and Pattern Recognition

Zia AI can now analyze CRM dashboard metrics and visualizations to provide intelligent insights about performance changes. This feature automatically identifies patterns in KPIs and charts, helping users understand the story behind their data without manual analysis.

 
Zoho CRM

Zia Digital Employees with Role-Based Permissions

Zia Agents can now be configured as digital employees with specific roles and permissions within Zoho CRM. This advancement allows organizations to deploy AI assistants that operate within defined security boundaries while handling routine tasks autonomously.

 
Zoho CRM

Sandbox Environment Supports Zoho Desk Integration

The Zoho CRM sandbox environment now includes full support for testing Zoho Desk integrations. This enhancement enables developers and administrators to safely test cross-application automations and workflows before deploying them to production environments.

 
Zoho Social

Q1 2026 Feature Roundup and Improvements

Zoho Social has released a comprehensive summary of all new features and enhancements introduced during the first quarter of 2026. The updates focus on improving social media management capabilities and streamlining content publishing workflows for marketing teams.
